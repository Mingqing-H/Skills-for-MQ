# 呼应式构图

## 适用场景

品牌视觉系统、系列产品、书店与文化活动、需要统一感和节奏感的海报。

## 必填变量

`{{形状呼应}}`、`{{色彩呼应}}`、`{{方向呼应}}`、`{{主标题}}`、`{{信息区}}`、`{{画幅}}`

## 设计机制

- 明确写出哪些元素互相呼应，以及呼应发生在形状、颜色、方向、大小或材质中的哪个维度。
- 让呼应元素分布在不同区域，通过重复与变奏形成首尾联系。
- 限制为 2–3 组呼应关系，避免到处复制同一元素。

## 提示词骨架

```text
设计一张{{画幅}}海报，采用呼应式构图，建立以下{{两/三}}组关系：

1. 形状呼应：{{区域A元素}}与{{区域B元素}}都呈{{共同形状}}，内容不同、轮廓相似。
2. 色彩呼应：{{区域C色块}}与{{区域D色块}}共享{{色相}}，通过{{明度/大小}}变化形成变奏。
3. 方向呼应：{{线条或主体A}}与{{线条或主体B}}都朝{{方向}}延伸，平行但不机械复制。

主标题“{{主标题}}”放在{{标题位置}}；{{信息区}}承载“{{必要信息}}”。背景{{背景}}，材质{{材质}}，让重复关系形成节奏而非装饰噪音。

避免：只写“互相呼应”、呼应维度不明确、重复元素完全一样、超过三组关系、标题与呼应元素争抢焦点。
```

## 质量检查

- 每组呼应的元素和维度是否都写清。
- 呼应是否有变化而非机械复制。
- 是否控制在三组以内。
