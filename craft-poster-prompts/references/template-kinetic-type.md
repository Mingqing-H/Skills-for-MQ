# 动态块状字体摄影海报

## 适用场景

运动、生活方式、青年文化、夏日活动、街头品牌、课程招募。

## 必填变量

`{{主体}}`、`{{动作}}`、`{{场景}}`、`{{主文本}}`、`{{强调色域}}`、`{{画幅}}`

## 设计机制

- 满版明亮摄影作为场域。
- 对角线动作和夸张裁切制造速度。
- 超大紧缩块状字体与主体遮挡，形成第二个图形体。
- 微型文案只作为节奏，不伪造信息。

## 提示词骨架

```text
创作一张{{画幅}}的高能量编辑摄影海报。{{主体}}在{{场景}}中进行{{动作}}，画面充满{{光线与天气}}。

构图采用强对角线，主体从{{入画方向}}切入，使用{{广角或长焦}}与大胆近距离裁切。保留动作最有张力的肢体和{{关键道具}}，背景以{{强调色域}}形成大面积干净色域。

主文本“{{主文本}}”使用超大、紧缩、厚重的{{文字颜色}}块状无衬线字体，跨越画面并被主体局部遮挡，但关键字必须完整可识别。辅助文字只使用“{{精确副文本}}”，以小型簇状排版放在{{位置}}。

加入{{颗粒或印刷处理}}，保持阳光、清爽、年轻、有速度感。

避免：动作僵硬、字体完全遮挡主体、文字不可读、背景杂乱、多个字体家族、过度霓虹、假编号和乱码。
```

## 质量检查

- 动作线、标题方向和裁切是否互相加强。
- 主体与文字谁是第一视觉是否明确。
- 辅助文案是否真实且足够少。
