# 体育与活动事件海报

## 适用场景

比赛、演唱会、音乐节、峰会、发布会、节庆活动、倒计时。

## 必填变量

`{{事件名}}`、`{{主角或阵容}}`、`{{动作}}`、`{{日期}}`、`{{地点}}`、`{{氛围}}`、`{{画幅}}`

## 设计机制

- 用单个关键动作和方向性光线制造“事件正在发生”。
- 标题、日期、地点形成稳定信息塔。
- 可用速度轨迹、舞台光、烟雾或观众轮廓增强能量，但不遮挡主角。

## 提示词骨架

```text
设计一张{{画幅}}的{{事件类型}}官方宣传海报，事件名“{{事件名}}”，氛围{{氛围}}。

主视觉为{{主角或阵容}}正在{{动作}}的决定性瞬间，采用{{机位}}和{{镜头}}，运动方向从{{起点}}指向{{终点}}。背景以{{场景元素}}建立规模感，并用{{能量机制}}强化现场能量。

信息层级只包含：“{{事件名}}”“{{日期}}”“{{地点}}”“{{必要补充信息}}”。文字清晰、可读，不生成虚构赞助商、队徽、票价或赛事标志。

色板{{色板}}，灯光{{灯光}}，材质{{材质}}，保留{{安全边距}}用于渠道裁切。整体具有电影 key art 的能量和真实活动海报的信息纪律。

避免：多人脸混合、肢体重复、错误球衣或器材、虚构 logo、信息被烟雾遮挡、动作无重心、视觉焦点分散。
```

## 质量检查

- 日期、地点、事件名是否一眼找到。
- 动作是否有明确方向、重心和瞬间感。
- 是否避免未经提供的官方标识。
