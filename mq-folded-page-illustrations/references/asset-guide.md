# 参考资产选择规则

正式参考资产统一放在 `assets/mq-reference-assets/`。`legacy-xiaohei-examples` 属于原技能的旧小黑案例目录，不得作为 MQ 技能的默认输入。

## 资产职责

- `01-character-master-sheet.png`：角色总基准。每次出现折页员都必须使用。
- `02-character-turnaround.png`：四视图、折面结构和缩小轮廓。侧面、背面、旋转或非常规视角时使用。
- `03-action-vocabulary.png`：折、夹、压、量、抽取、归档等动作。需要明显肢体动作时使用。
- `04-line-and-type-style.png`：石墨双线宽、钴蓝工程线、窄体硬笔中文和 MQ 朱印。带中文批注时优先使用。
- `05-example-fold-shortcut.png`：路径、步骤精简和“把长流程折短”的构图参考。
- `06-example-filter-noise.png`：过滤、筛选、噪声与有效信号的构图参考。

## 选择方法

常规正文配图：

1. 必选 `01-character-master-sheet.png`。
2. 有中文批注时加 `04-line-and-type-style.png`。
3. 根据主题在 `02`、`03`、`05`、`06` 中最多选 1 张。

单次输入最多 3 张参考图，避免角色与构图过度锁死。

## 使用边界

- `01` 与 `02` 用于身份一致性，可以高强度遵循。
- `03` 用于动作语言，只借动作原则，不复制整页布局。
- `04` 用于线条、颜色和文字，不复制其中的三个微场景。
- `05` 与 `06` 只作低频构图校准。除非用户明确要求复刻，否则不得复制物件、路径和标注。
- 不得从旧小黑案例继承角色、实心黑造型、橙色流程线或旧手写字体。
