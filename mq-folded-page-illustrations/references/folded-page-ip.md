# 折页员 IP

## 角色定义

折页员是 MQ 中文正文配图的固定视觉 IP。它像一张被临时折成工作人员的纸页：安静、认真、略显笨拙，擅长把复杂问题折叠、展开、归档和校准。

默认每张图都出现折页员，并让它承担核心动作。去掉角色后，如果隐喻仍可完整成立，说明角色太装饰。

## 固定造型

必须保持：

- 米白色单张折纸身体，纵向、不对称、略向一侧倾斜。
- 左上有一个明显折角，右侧有一处较小的内折结构。
- 面部只有一条黑色横向窄视窗；无圆眼、眉毛、嘴巴和表情符号。
- 右上夹一枚黑色金属回形针；位置与比例稳定。
- 两条细黑手臂、两条细黑腿，关节与手掌只作简写。
- 胸口有小型朱红方印；内部 `M` 在上、`Q` 在下，近看清楚，远看仍像印章。
- 纸身允许极轻的米白铅笔铺色，不得变成黄色纸张或真实纸纹。

生成时使用 `assets/mq-reference-assets/01-character-master-sheet.png` 作为 reference image，并明确要求保持角色身份与比例。

## 性格

- 冷静、克制、可靠，不卖萌。
- 像认真执行任务的文档管理员或现场校对员。
- 动作有一点笨拙，判断却准确。
- 幽默来自“纸页角色在认真处理抽象系统”，不来自夸张表情。

## 动作语言

优先使用与纸张身份有关的动作：

- 折叠、展开、压平、夹住、归档、抽取。
- 拉线、校准、丈量、标记、拼接、开箱。
- 用身体充当门、挡板、漏斗、桥面或临时容器。
- 搬运、推、扛、拽、托、接、守、修补。

动作必须服务文章含义，不要每张都强调“折纸”。

## MQ 朱印

- 只出现在角色胸口，不作为页面水印或角标。
- 使用朱红 `#C9412E` 附近色。
- 保持小、方、略有断墨。
- `M` 与 `Q` 可共享边框或部分笔画，但两个字母都必须可辨。
- 不使用标准无衬线字体，不放大成品牌 Logo。

## 禁止

- 不得变成黑色实心怪物、动物、机器人、信封或购物袋。
- 不得增加眼睛、嘴、眉毛、服装、帽子或复杂配件。
- 不得丢失回形针、黑色窄视窗或 MQ 朱印。
- 不得过度拟人、圆润可爱、Q 版或表情包化。
- 不得画成规整矢量图标或精致 3D 折纸。
