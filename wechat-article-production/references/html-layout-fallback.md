# HTML Layout Fallback

Use this when no dedicated WeChat layout skill is available.

## Rules

- Preserve visible text exactly.
- Use simple HTML and inline or embedded CSS.
- No JavaScript, forms, iframes, tracking pixels, or remote fonts.
- Use a top-level `<section>` wrapper.
- Keep mobile reading comfortable: 15-16px body, line-height 1.75-2.
- Image placeholders may be styled as blocks but text must remain visible.

## Default Style

- Background: warm white or soft paper tone.
- Text: dark charcoal.
- Accent: one restrained teal/blue/green or warm orange.
- Headings: bold with left accent line or subtle background.
- Quotes/key sentences: light background card with border.
- Tables: scrollable, readable, subtle header fill.
- Captions/source notes: small muted centered text.

## Validation

Before finalizing:

- Count source image references and HTML image slots.
- Compare heading order.
- Check source paragraphs appear in order.
- Confirm no decorative labels add visible editorial content unless present in source.
